D1 Revised Project Statement
Locked contribution — one sentence
An AI agent autonomously discovered that switching from linear regression to Random Forest — not feature engineering — was the dominant driver of prediction accuracy for daily solar irradiance across three climatically diverse US cities, reducing validation RMSE by 43% from baseline.

What the project is: A Karpathy-style AutoResearch loop in R where a Codex agent iterates on train.R, runs experiments via a frozen run.R executor, and logs every result to experiments.json. The agent may only modify the model file; evaluation, data, and logging are frozen to ensure comparability.

What the agent discovered: Across 10 controlled experiments, feature additions under linear regression produced marginal or negative gains. Switching to Random Forest with the same feature set produced the largest single-step improvement in the log (−24% in one run). The agent identified this through a controlled experiment, not random search.

Why it matters: The project demonstrates that an autonomous agent can produce interpretable, reproducible research findings — not just score improvements — when given a disciplined experimental framework with frozen evaluation and mandatory logging.
